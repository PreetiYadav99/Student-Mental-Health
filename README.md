Student Mental Health Analysis Dashboard
📌 Project Overview

The Student Mental Health Analysis Dashboard is a data analytics project developed to analyze factors affecting students' mental well-being using interactive visualizations. The dashboard provides insights into stress levels, academic pressure, sleep patterns, study hours, financial stress, depression, anxiety, and other lifestyle factors that influence students' mental health.

The project leverages Power BI for data visualization and dashboard creation, enabling users to explore trends, identify high-risk groups, and make data-driven decisions for improving student well-being.

🎯 Objectives
Analyze the mental health status of students.
Identify major factors contributing to stress, anxiety, and depression.
Study the relationship between academic pressure and mental health.
Examine the impact of sleep duration, study hours, and lifestyle habits.
Build an interactive dashboard for easy exploration of insights.
Support educational institutions in understanding student wellness trends.
